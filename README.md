# SmartPetHome

SmartPetHome is a pervasive and mobile system for intelligent pet feeding monitoring,
developed in the scope of PAMO and CMOV courses at Universidade Fernando Pessoa.

## Components
- Android Application (Jetpack Compose, Room, Firebase)
- IoT Sensor Node (Arduino + Ultrasonic Sensor)
- Cloud Backend (Firebase, REST API)
- IEEE Scientific Paper (CMOV)
